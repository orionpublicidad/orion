    CARACTERISTICAS GENERALES DEL HARDWARE REQUERIDO

  • Para el manejo de Orión Software sobre el sistema operacional Windows, le sugerimos proveer su servidor con un procesador Intel de cualquier velocidad con los estándares normales.
  • Para estaciones de redes locales de trabajo superiores a cuatro (4), recomendamos utilizar un servidor con un sistema operativo especializado de red como Windows 2000 Server o NT Server.
  • Sugerimos la configuración "TIPICO HOY", por tratarse de un modelo estándar.
  • En la configuración de los servidores, no sugerimos procesadores AMD para el funcionamiento del software.

    • REQUERIMIENTOS MINIMOS PARA FUNCIONAMIENTO DE ORION SOFTWARE

  • Sistema operativo Windows XP en adelante. Tipo de sistema operativo de 32 bits
  • Procesador 2.0 GHz.
  • Unidad de CD ROM
  • Capacidad de disco duro 160 Gb
  • Memoria 2Gb.
  • Si el sistema se instalara en red, los protocolos de la red deben de ser TCP/IP con dirección fija.
  • Resolución de los monitores mayor a 256 colores.
  • El servidor debe poseer un dispositivo de CD ROM (quemador) o puertos USB. Esto con el fin de hacer los backup o copia de seguridad
  • No instalar en sistemas de visualización de PC tales como Between, MagicMirror entre otros.
